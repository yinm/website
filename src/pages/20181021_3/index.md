### 横並びにする要素の幅を指定する
- `flex-basis: 横幅の%;`で指定する
  - `flex: 0 0 横幅の%;`でも可能 (ショートハンドプロパティ)
- 100%になった時に別の行にするには、`flex-wrap: wrap;`を指定する
